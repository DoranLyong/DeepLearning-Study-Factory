# PyTorch TensorBoard Tutorial 


텐서보드 사용법 익히기 


### 설치: 
```bash 
pip install tb-nightly
```


### 실행: 
```bash 
~$ tensorboard --logdir <로그가 저장되는 폴더>

(ex)
~$ tnesorboard --logdir runs

```
